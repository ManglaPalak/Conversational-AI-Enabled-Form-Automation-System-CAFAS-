# AI Form Filling Prototype

## Overview
This project demonstrates a voice-enabled, multilingual, schema-driven form filling system using the Web Speech API.

## Features
- Voice input for filling form fields
- Multilingual support (English, Hindi, French, Spanish)
- Dynamic form generation from JSON schema
- Basic ambiguity handling (e.g., "next Monday" for dates)
